🌲 Sean's Cali Fitness Tracker – Full App Specification

🧭 Background & Purpose
This app is designed for daily logging and progress tracking of my calisthenics-based workout routine using the CaliMove program. I follow:

Move 2.0 (Mobility Program) – ~210 days
Complete Calisthenics Level 2 – ~130 days
Additional handstand skill work and bodyweight strength training
Workouts are done in a gym or minimal equipment setup, usually on phone.
Tracking is done during workouts (not after) to stay focused and log set-by-set.

Key goals:

See which Mobility/Strength Day I last completed
Log each exercise and set individually
Track bodyweight over time
Quickly check past workouts
Maintain a clean, motivating mobile UI
🧩 Core Features

🟢 Log daily workouts
🟢 Track Mobility, Handstand, Strength training
🟢 Support both rep-based and time-based exercises
🟢 See previously completed CaliMove day for Mobility and Strength
🟢 Record different reps per set
🟢 Bodyweight input
🟢 View past workouts in Progress Tracker
🟢 Offline-friendly (PWA)
🎨 Visual Design

🧪 Data Structure

interface Exercise {
  name: string;
  isTimeBased: boolean;
  sets: {
    value: number; // seconds or reps
  }[];
}

interface Workout {
  date: string;
  weight: string;
  mobility: {
    dayNumber?: number;
    completion: 'not-completed' | 'half-session' | 'full-session';
  };
  handstand: {
    exercises: string[]; // Names only
  };
  strength: {
    dayNumber?: number;
    exercises: Exercise[];
  };
}
